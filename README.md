# My Music Store 🎶

A lightweight website for streaming and selling music tracks.

## Features
- Stream songs directly in the browser
- Simple "Buy" buttons for each track
- Clean, responsive design

## How to Use
1. Open `index.html` in your browser.
2. Click play to stream tracks.
3. Use the Buy button to purchase (currently placeholder).
